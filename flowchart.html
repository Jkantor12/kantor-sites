Python + Flask Backend

Built → thekantorco.com, stem-co store, /hire page, 27+ business sites

Python SDK for placing bracket orders, reading positions, and streaming market data in the trading bot.

Trading Bot — Scan → Trade → Monitor Pipeline

Alpaca market data (235 stocks, every 25s)
→
Scanner checks RSI, EMA, momentum, sentiment
→
Signal queued if 2+ signals agree
→
Bracket order placed (stop -2.5%, target +3%)
→
Monitor checks every 5 min — breakeven at +1.5%, extend to +5% on momentum
→
Pushover / email notification on close

APIs & External Services — What Each One Powers
Anthropic (Claude)
Used by: TopPick, Claude Code
Writes affiliate articles in Content Farm. Also the AI that built this entire system via Claude Code CLI.
CJDropshipping API
Used by: The Stem Co.
Sources products and auto-fulfills orders. Bot authenticates with CJ_API_KEY + CJ_EMAIL.
PayPal IPN
Used by: Stem Co., /hire
Webhook fires to Railway on every sale. Routes by item_name to either CJ order or site build.
Printify API
Used by: JaMadeGoods
Receives design uploads, creates products, publishes to Etsy. Shop ID: 28027468.
Etsy API (OAuth)
Used by: Digital Vault, Template Studio, Drop Factory
Lists digital products, preset packs, templates, and dropship items on JaMadeGoods Etsy shop. x-api-key requires full keystring:secret.
Google Places API
Used by: Lead Finder
Searches Denver businesses by category. Free up to $200/month credit.
Twilio REST API
Used by: Lead Finder
Sends SMS to local businesses. From +17372583742. A2P 10DLC compliance in review.
Alpaca Markets API
Used by: Trading Bot
Places bracket orders, reads positions, streams market data. Paper trading mode.
Kalshi API
Used by: Prediction Markets
Reads prediction market contracts, places YES/NO bets when edge ≥ 5%.
Amazon Associates
Used by: TopPick
Affiliate tag kantorland-20 embedded in all article links. Need 3 sales to unlock full API.
Pinterest Ads API
Used by: Ad Studio
Running 3-ad campaign for The Stem Co. $7/day, Consideration objective. Launched 2026-06-25.
Gmail SMTP
Used by: Weekly Recap, Lead Finder, Trading Bot
Sends recap emails, order confirmations, and trade notifications from jbot0012@gmail.com.
Pushover API
Used by: Trading Bot
Sends push notifications to your phone on trade entries, exits, and milestones.
GitHub API / Pages
Used by: Web Studio, TopPick
Git push deploys new sites and articles to thekantorco.com instantly via GitHub Pages.
Railway API
Used by: Habitat dashboard
Auto-deploys both Railway services on every push to their respective repos.
